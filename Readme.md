# Go Go Gopher

A little platformer game written in Go.

The game is based on the [Ebiten](https://github.com/hajimehoshi/ebiten) game engine
by [Hajime Hoshi](https://github.com/hajimehoshi).

## Compile the game

````bash
go genereate ./...
go build .
````
